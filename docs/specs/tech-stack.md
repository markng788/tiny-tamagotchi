# Tech Stack

Vanilla HTML, CSS, and JavaScript only. No frameworks, no build tools, no dependencies.

## Architecture

Single-page application (`index.html`). All logic in one JavaScript file (`app.js`). All styles in one CSS file (`style.css`).

## State Persistence

localStorage. Save pet name, stats, state, and evolved timer on every stat change.

## File Structure

| File        | Purpose           |
|-------------|-------------------|
| index.html  | Main page         |
| app.js      | All game logic    |
| style.css   | All styles        |
| docs/       | All spec documents |

## Coding Standards

- No external libraries
- All variables in English
- Comments in English
- Compatible with Chrome, Firefox, and Edge
